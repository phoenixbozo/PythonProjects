----------------------------------------------------------

1. **Installer Python :**
   - Rendez-vous sur le site officiel de Python (https://www.python.org/) pour télécharger la version compatible avec votre système d'exploitation.
   - Suivez les instructions d'installation pour installer Python.

2. **Installer les bibliothèques :**
   - Ouvrez une fenêtre de terminal ou de ligne de commande.
     ```
     pip install -r requirements.txt
     ```

     ou

   - Utilisez la commande pip pour installer les bibliothèques requises :
     ```
     pip install matplotlib
     pip install tkcalendar
     ```

3. **Exécuter le programme Python :**
     ```
     python PyWeightTracker.py
     ```
     ou
     ```
     python3 PyWeightTracker.py
     ```
