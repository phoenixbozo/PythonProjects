Preparing the machine for running Python code
---------------------------------------------

Steps to prepare the computer:

1. **Install Python:**
   - Go to the official Python website (https://www.python.org/) to download the version compatible with your operating system.
   - Follow the installation instructions to install Python.

2. **Install libraries:**
   - Open a terminal or command line window.
     ```
     pip install -r requirements.txt
     ```

     or

   - Use the pip command to install the required libraries:
     ```
     pip install matplotlib
     pip install tkcalendar
     ```

3. **Run the Python program:**
     ```
     python PyWeightTracker.py
     ```
     or
     ```
     python3 PyWeightTracker.py
     ```
